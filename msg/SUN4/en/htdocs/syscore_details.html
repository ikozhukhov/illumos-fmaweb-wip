The Message ID:  %MSGID% indicates a device connected to the hostbridge bus is faulty and requires Service. The system will attempt to disable the faulty device and allow the rest of the system to continue operation.

Use the fmdump -v -u EVENT_ID command to determine the faulty components as shown in the example below. 
# fmdump -v -u 6999fd18-1feb-499f-acf7-cceb78deb18c
TIME                 UUID                                 SUNW-MSG-ID
Nov 17 20:46:06.3294 6999fd18-1feb-499f-acf7-cceb78deb18c SUN4-8000-FC
  100%  fault.io.fire.hbus
         FRU: hc://product-id=SUNW,Sun-Fire-T200/component=IOBD
        rsrc: hc:///ioboard=0/hostbridge=0/pciexrc=0
Use the fmdump -eV -u EVENT_ID command to collect the ereports associated with this problem.

Have this information available when contacting Sun Services for a repair.
