The Message ID:  %MSGID% indicates a problem with the hardware and/or driver software in the PCI subsystem. The most likely faulty components, as shown with fmdump, should be repaired in order to restore the system.

If the most likely suspect is a package then run pkgchk on the package(s). Check for updates and patches for suspected package(s). Contact Sun Service if no problems are found with package or no updates exist.

Potentially defective software is identified in the fmdump output as follows:

    FRU: pkg:///SUNWcakr

Potentially faulty hardware is identified in the fmdump output as follows:

    FRU: hc://product-id=SUNW,Sun-Fire-280R/motherboard=0

If there are multiple FRU's suspected, the FRU with the highest percentage probability should be replaced first.

Note: You may also be requested to provide a copy of the files in the /var/fm/fmd directory and also a copy of the output from the fmadm config command.
