The Message ID:   %MSGID% indicates one or more uncorrectable memory errors occurred. The memory pages associated with the uncorrectable errors have been retired. However, the system is at risk of incurring additional uncorrectable errors. There may be two memory DIMMs associated with the error. Replace both DIMMs as soon as possible.

Use the command fmdump -v -u <EVENT_ID>  with the  <EVENT_ID>  from the console message to locate the faulty DIMMs. Depending on the platform the fault was diagnosed on, the location of the faulty DIMMs will be displayed in one of two ways. For some platforms, the output of fmdump will appear as follows: 

fmdump -v -u d5b2eeda-ff44-622f-b794-969dbd2082fe  
TIME                 UUID                                 SUNW-MSG-ID
Sep 14 09:54:33.7068 d5b2eeda-ff44-622f-b794-969dbd2082fe SUN4U-8000-E2
   95%  fault.memory.bank
         FRU: mem:///component=MB/CMP0/CH0:R0/D0/J0601,R0/D1/J0701
        rsrc: mem:///component=MB/CMP0/CH0:R0/D0/J0601,R0/D1/J0701

In this example, the location of the two DIMMs are: 

MB/CMP0/CH0:R0/D0/J0601
MB/CMP0/CH0:R0/D1/J0701

The alternate format for the output of fmdump for this fault is: 

fmdump -v -u  b277ec2e-55b6-ea0b-8c88-aaff3ba7fa9a 
TIME                 UUID                                 SUNW-MSG-ID
Jun 21 23:50:16.6635 b277ec2e-55b6-ea0b-8c88-aaff3ba7fa9a SUN4V-8000-E2
95%  fault.memory.bank
    Problem in:mem:///unum=MB/CMP0/BR1:CH1/D0/J1601
       Affects:mem:///unum=MB/CMP0/BR1:CH1/D0/J1601
          FRU: hc://:serial=00CE01062351032B66:
            part=371-2143-01 Rev01//motherboard=0/chip=0/branch=1/
            dram-channel=1/dimm=0
       Location: MB/CMP0/BR1: CH1/D0/J1601
95%  fault.memory.bank
    Problem in:mem:///unum=MB/CMP0/BR1:CH0/D0/J1401
       Affects:mem:///mem:///unum=MB/CMP0/BR1:CH0/D0/J1401
       FRU: hc://:serial=00CE01062351032E73:
          part=371-2143-01 Rev 01//motherboard=0/chip=0/branch=1/
          dram-channel=0/dimm=0
      Location: MB/CMP0/BR1: CH0/D0/J1401

In this example the FRU includes a part number field: 

part=371-2143-01 Rev01

This is the part number of the faulty DIMM. The locations of the two DIMMs in this example are specified in the “Location” field for each fault DIMM: 

MB/CMP0/BR1: CH1/D0/J1601
MB/CMP0/BR1: CH0/D0/J1401

Refer to the Service Manual or the Service Label attached to the server chassis to find the physical location of the DIMMs. Some platforms also provide fault indicators for individual DIMMs to validate the location. Once the DIMMs have been replaced, use the Service Manual and Product Notes for instructions on clearing the fault condition and validating the repair action.
